# Student Grade Calculator

This project calculates student grades based on marks entered by the user.

## Features
- Uses if-elif-else conditions
- Uses while loop for validation
- Uses functions
- Displays encouraging messages
- Validates marks between 0 and 100

## Technologies Used
- Python

## Setup Instructions
1. Install Python
2. Open CMD
3. Navigate to project folder
4. Run:
python grade_calculator.py

## Grading Logic
- A : 90-100
- B : 80-89
- C : 70-79
- D : 60-69
- F : 0-59
